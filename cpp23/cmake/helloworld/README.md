# cpp23-cmake-helloworld

基于cmake构建工具cpp23标准的HelloWorld项目工程模板

## 创建命令

```
xlings new --template cpp23-cmake-helloworld
```

## 基本使用

```bash
mkdir build && cd build
cmake ..
make
./helloworld
```
